# Spacing

USWDS spacing tokens are based on a base-8 scale. The token number corresponds to multiples of 8px, with some half-step values available.

## Usage Guidelines

- Use spacing tokens instead of hardcoded pixel values — this ensures consistency across all components
- Prefer multiples of 2 for most spacing needs — tokens 1 through 6 cover the majority of use cases
- Use half-step tokens (05, 105, 205) sparingly for fine-tuning only
- Consistent spacing is a key part of visual rhythm — components and page templates should draw from the same scale
